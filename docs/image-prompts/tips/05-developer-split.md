# Contextual Tip: Developer Split

## Context

**When Shown**: After payout tip is dismissed
**Audience**: Hosts
**Purpose**: Explain the developer fee contribution slider

---

## Primary Prompt

```
Modern flat illustration, vertical mobile format. A pie chart or split visualization showing money being divided: larger portion going to host (green), smaller portion to developer (orange). Slider or adjustment control below the chart. Heart or support symbols around the developer portion. Orange and green color palette with subtle glow effects. Clean minimalist design, conveying voluntary contribution and sustainability. No text, high contrast, vector-style artwork. --style flat --ar 2:3
```

---

## Alternative Prompt

```
Revenue sharing illustration. Two stacks of coins: a tall green stack (host's portion) and a smaller orange stack (developer fee). An adjustable slider between them showing the split can be changed. Heart or thumbs-up icon near the developer stack. Orange and green gradients creating a sense of fairness and choice. Modern flat design, vertical orientation, conveying "support the app, you choose how much". No text, professional quality. --style flat --ar 2:3
```

---

## Key Elements

- Split visualization (pie chart or stacks)
- Host portion (larger, green)
- Developer portion (smaller, orange)
- Adjustable slider/control
- Heart or support symbols
- Orange and green color coding
- Voluntary contribution feeling
- Fair, transparent design
