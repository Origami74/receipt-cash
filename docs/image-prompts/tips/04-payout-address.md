# Contextual Tip: Payout Address

## Context

**When Shown**: First time on payment setup screen OR when user focuses on address input
**Audience**: Hosts
**Purpose**: Explain where to enter Lightning address or Cashu payment request

---

## Primary Prompt

```
Modern flat illustration, vertical mobile format. A smartphone with two input field options stacked: top field showing Lightning address format (user@domain.com) with orange lightning bolt icon, bottom field showing Cashu payment request format (creqA...) with green ecash/nut icon. Arrows showing money flowing from the app to both destinations. Orange and green color palette with subtle glow effects. Clean minimalist design, conveying flexible payment options. No text, high contrast, vector-style artwork. --style flat --ar 2:3
```

## Key Elements

- Two input field options (stacked or side-by-side)
- Lightning address: user@domain.com format with ⚡ icon
- Cashu request: creqA... format with 🥜 or ecash icon
- Money flow visualization
- Wallet/destination icon
- Orange (Lightning) and green (Cashu) color coding
- Flexible choice feeling
- Clear, simple design
